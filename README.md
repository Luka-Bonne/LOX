# LOX

LOX - это игра-аналог старой аркадной игры Q*bert, но с изменённым дизайном. Действия новой игры происходят в космосе, а главным героем является LOX - неизвествое существо в красном плаще. Цель игры - поситеть все поля карты, перекрасив их в новый цвет.

Правила пользования:
1. Для перемещение персонажа нужно использовать сочетания клавиш:
   - W + A - перемещение на верхнее левое поле,
   - W + D - перемещение на верхнее правое поле,
   - S + A - перемещение на нижнее левое поле,
   - S + D - перемещение на нижнее правое поле.
2. При первом посещении куба он становится закрашенным, при следующем посещении этого же куба он теряет свой цвет. Для того, чтобы снова закрасить куб, нужно уйти с него и посетить заново.
3. На поле в процессе игры появляются мобы-слизняки, при контакте с ними персонаж умирает и перерождается, откатываясь на начальное поле.
4. Просто как плюс игру можно поставить на стоп с помощью кнопки в верхнем правом углу. Кнопка в верхнем левом углу останавливает воспроизведение музыки.
